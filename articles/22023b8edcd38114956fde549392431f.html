ChatGPT Adds Cloud Storage Connectors

Published: Sun, 29 Jun 2025

ChatGPT has added connectors for Google Drive, Dropbox, SharePoint, and Box for paying users. These connectors allow ChatGPT to access files stored in these cloud services. Users can access these connectors through their profile settings and can search these sources during chats by selecting the appropriate connector.
