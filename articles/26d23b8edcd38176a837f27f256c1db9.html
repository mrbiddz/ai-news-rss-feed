Claude Introduces File Creation and Editing

Published: Sun, 14 Sep 2025

Claude now allows users to create and edit files directly within the platform, including Excel spreadsheets, documents, PowerPoint slides, and PDFs. This feature is currently available in Max, Team, and Enterprise plans, with Pro users getting access in the coming weeks. Users need to enable the 'upgraded file creation and analysis' feature in settings to use this functionality.
