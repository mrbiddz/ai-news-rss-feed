OpenAI Launches Codeex CLI and Explores Coding Tool Acquisition

Published: Sat, 19 Apr 2025

OpenAI rolled out Codeex CLI, an open-source coding tool for terminal use. Additionally, rumors suggest OpenAI is in talks to acquire Windsurf, an AI coding agent interface (similar to Cursor), for approximately $3 billion. They reportedly considered buying Cursor first but found its $10 billion valuation too steep.
