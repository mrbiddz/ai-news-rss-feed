Ideogram Launches Real-time Video Generation

Published: Sun, 14 Sep 2025

Ideogram has rolled out a real-time video generation feature that allows users to edit an image and see the corresponding video update in near real-time. While there is a slight delay, the feature enables users to create animations by manipulating elements in the source image. The feature is currently being rolled out gradually to users.
