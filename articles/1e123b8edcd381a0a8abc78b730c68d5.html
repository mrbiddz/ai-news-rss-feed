OpenAI Planning to Release Open Model

Published: Sun, 27 Apr 2025

OpenAI is planning to release an open model around June that will be available to download at no cost and won't be gated behind an API. The model is targeting performance superior to open models from Meta and Deepseek. This new model will reportedly be able to call upon other models, potentially making calls to the OpenAI API to access larger models for more complex computational tasks. It's still unclear whether it will be able to do web search or image generation.
