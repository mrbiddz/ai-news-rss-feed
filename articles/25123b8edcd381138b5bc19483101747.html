Skywork AI Releases Matrix Game 2.0

Published: Sun, 17 Aug 2025

Skywork AI released Matrix Game 2.0, which appears to be an open-source version of Google's Genie 3. The technology allows users to enter and navigate 3D worlds generated from a single input image. It works by generating a full 360-degree panorama first, then extending it temporally with camera control before lifting it to 3D. This creates consistency when looking around the virtual environment. However, it requires significant computing power - an Nvidia GPU with at least 24GB of memory.
