Ideogram Introduces Style Feature

Published: Sun, 14 Sep 2025

Ideogram (pronounced 'ideogram', not 'idogram') has released a new style feature that allows users to provide a style reference image or choose from pre-created styles when generating images. The system will then create images matching the selected style based on the user's prompt.
