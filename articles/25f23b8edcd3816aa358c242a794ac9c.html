Cling Introduces First and Last Frame Animation Feature

Published: Sun, 31 Aug 2025

Cling has introduced a powerful animation feature that allows users to provide the first and last frames of a video, with the AI generating all frames in between to create a smooth animation. The tool is particularly effective for morphing effects, as demonstrated with an example of a person transforming into a wolf. According to the presenter, this is the most impressive start frame/end frame video tool available.
