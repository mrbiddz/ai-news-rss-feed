Perplexity Launches Comet Browser

Published: Sun, 13 Jul 2025

Perplexity has launched a new AI-powered web browser called Comet. The browser is currently available to Perplexity Max subscribers ($200/month) with invite-only access rolling out over the summer. Comet is based on Chrome but integrates Perplexity's AI assistant directly into the browsing experience. Key features include screen awareness (the assistant can see and analyze what's on screen), price comparison shopping, automatic actions like posting to social media, email checking, and webpage summarization. The browser represents a fundamentally different way of using the internet, with less manual browsing and more AI assistance.
