Microsoft Updates 365 Copilot with Agentic Features

Published: Sun, 27 Apr 2025

Microsoft has updated 365 Copilot with more agentic features, including AI-powered search, a new create experience, Copilot notebooks, and a new agent store. The update includes specialized agents like researcher and analyst that are optimized for specific tasks. The researcher agent can do deeper dive research, while the analyst agent can analyze data from Excel, Word documents, and other sources. Most users will be able to access these features starting in late May.
