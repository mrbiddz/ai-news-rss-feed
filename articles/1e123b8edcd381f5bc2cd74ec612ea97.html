OpenAI Expresses Interest in Buying Google Chrome

Published: Sun, 27 Apr 2025

In a court case against Google, OpenAI stated that they would bid to buy Google's Chrome browser if it became available for sale. With courts considering breaking up Google due to monopoly concerns, both OpenAI and Perplexity have expressed interest in acquiring Chrome. The presenter speculates that OpenAI might want to build an AI-first browser using Chrome's existing foundation.
