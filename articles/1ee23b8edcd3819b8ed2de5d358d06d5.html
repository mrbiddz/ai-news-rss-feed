Windsurf Releases Wave 8 Update with New Features

Published: Sat, 10 May 2025

Windsurf released Wave 8, a major update with numerous new features including Windsurf Reviews for reviewing GitHub pull requests, the ability to add knowledge from Google Docs and API documentation, conversation sharing, Teams deploy feature, and updated analytics. The company is rolling out these features gradually rather than all at once.
