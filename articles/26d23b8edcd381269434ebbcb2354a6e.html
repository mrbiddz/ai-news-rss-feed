Nano Banana Integration with Leonardo and Adobe Photoshop

Published: Sun, 14 Sep 2025

OpenAI's Nano Banana image editing model is now available directly within Leonardo AI. Users can select Nano Banana as a model option when editing images. Additionally, Adobe has announced that Nano Banana will be integrated natively into Photoshop, allowing users to select it as one of the available models.
