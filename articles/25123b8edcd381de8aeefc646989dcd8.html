Microsoft Releases Copilot 3D

Published: Sun, 17 Aug 2025

Microsoft rolled out a new project called Copilot 3D, which allows users to turn any image into a 3D model. This tool can be used for creating 3D assets for game development, miniature modeling, 3D printing, web design, and virtual reality. It's freely available to anyone with a Microsoft account. The tool processes images into 3D models in about 15-20 seconds, which is significantly faster than previous image-to-3D technologies that took hours to process.
