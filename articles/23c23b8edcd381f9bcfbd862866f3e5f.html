Google Tests Web Guide for Reorganized Search Results

Published: Sun, 27 Jul 2025

Google is testing a new feature called Web Guide, an experimental AI-organized search results page. When users search for topics like 'how to solo travel in Japan,' it groups results into categories such as comprehensive guides, safety tips, accommodation options, etc. This feature needs to be turned on inside Google Labs.
