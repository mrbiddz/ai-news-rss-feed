Microsoft 365 Copilot Adds Anthropic Models

Published: Sun, 28 Sep 2025

Microsoft has added Anthropic's models to Microsoft 365 Copilot, allowing users to choose between GPT models and Anthropic models when using tools like Excel. This is notable because Microsoft owns 49% of OpenAI's non-profit side but is still diversifying by integrating competitor models.
