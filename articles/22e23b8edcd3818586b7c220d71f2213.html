V3 Adds Image-to-Video with Audio Feature

Published: Sun, 13 Jul 2025

V3 has launched a new feature that allows users to upload an image and convert it into a video with audio. Previously, V3 could only create videos without sound from images. This update enables users to get more consistent characters in their videos by using the same image across multiple generations. The feature is now available in the Gemini app. Additionally, V3 is now available globally for all Gemini Pro members ($20/month), including in India, Indonesia, Europe, and other regions.
