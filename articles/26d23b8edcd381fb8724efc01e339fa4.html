ChatGPT Adds Projects, Branching, and Memory Controls

Published: Sun, 14 Sep 2025

ChatGPT has rolled out several new features: Projects (folders to organize discussions) are now available in the free plan, users can upload more and larger files, and project-only memory controls allow for more tailored context. A new branching feature lets users create alternative conversation paths from any point in a chat.
