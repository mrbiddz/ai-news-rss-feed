Google's Nano Banana (Gemini Flash 2.5 Image) Released to Public

Published: Sun, 31 Aug 2025

Google has finally released Nano Banana (officially called Gemini Flash 2.5 image) to the public for free. This powerful image editing tool allows users to upload an image and transform it based on text prompts. New use cases include annotating real-world locations, creating isometric views from aerial images, restoring old photographs, and generating AR-like experiences. The tool has been integrated into multiple platforms including Higsfield, Crea, Scenario, and surprisingly, Adobe Firefly, where it's even set as the default model above Adobe's own Firefly models.
