Google Launches Open-Source Gemini CLI

Published: Sun, 29 Jun 2025

Google launched a new open-source Gemini CLI (Command Line Interface) that functions as an AI agent to write code directly from within a computer's terminal. It leverages various Gemini models and has received positive feedback from developers. Users can access the Gemini 2.5 Pro model for free with 60 model requests per minute and 2,000 requests per day. Being open-source allows others to build upon and improve the CLI.
