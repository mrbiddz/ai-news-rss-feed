Tencent Releases Hunan 3D 2.5 Model

Published: Sun, 27 Apr 2025

Tencent released a new model called Hunan 3D 2.5, which is a 10 billion parameter model (up from 1 billion parameters previously). The model features high-quality textures and an animation boost. Demo videos show impressive results, though the presenter notes these are likely cherry-picked examples.
