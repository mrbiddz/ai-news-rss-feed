Anthropic Releases Claude Opus 4.1

Published: Sun, 10 Aug 2025

Anthropic has released Claude Opus 4.1, which was briefly the most powerful AI coding model on the planet before being surpassed by GPT-5 a day later. The model was specifically optimized for coding tasks and scored 74.5% on the Swebench verified benchmark for software development, slightly below GPT-5's 74.9%. Claude Opus 4.1 is significantly more expensive than GPT-5, costing $15 per million input tokens and $75 per million output tokens, compared to GPT-5's $1.25 per million input tokens and $10 per million output tokens. Testing showed Claude Opus 4.1 created a more feature-rich fitness app from a single prompt compared to GPT-5.
