Anthropic Introduces Web Search in Claude API

Published: Sat, 10 May 2025

Anthropic has added web search functionality to their Claude API, allowing developers to build applications that can search the web directly through Claude. This means that tools like Windsurf that use Claude as their model for coding can now access the internet to help write better code.
