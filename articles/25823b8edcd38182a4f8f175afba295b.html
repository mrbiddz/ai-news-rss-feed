Google Docs Adds AI Voice Reading Feature

Published: Sun, 24 Aug 2025

Google Docs has added a new feature that allows documents to be read aloud using AI voices. Available under the Tools > Audio menu, users can select from different voice styles including narrator, educator, teacher, persuader, explainer, or coach. The feature appears to be available only on Google's paid pro and ultra plans.
