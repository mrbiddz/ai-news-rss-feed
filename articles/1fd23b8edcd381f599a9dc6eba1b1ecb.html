Microsoft Build Focuses on Developer Tools

Published: Sun, 25 May 2025

Microsoft Build focused heavily on developer tools, including Microsoft Discovery, an AI designed to help make scientific discoveries by building graphs of relationships between data. Other announcements included ChatGPT's image generator in Microsoft Copilot, AI updates to Windows apps like Paint and Notepad, GitHub Copilot with a new coding agent that can autonomously write code and fix issues, and the open-sourcing of GitHub Copilot in VS Code.
