Grock Launches Studio and Memory Features

Published: Sat, 19 Apr 2025

XAI's Grock released Grock Studio, adding code execution and Google Drive support in an interface similar to OpenAI's Canvas. It allows users to generate documents, code, reports, and browser games. Grock also introduced a memory feature that remembers past conversations to provide context for new chats, similar to OpenAI's recently launched memory feature. Both features are available on grock.com but not yet in the X integration.
