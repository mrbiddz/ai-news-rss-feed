Microsoft Adds Memory Feature to Copilot

Published: Thu, 17 Apr 2025

At Microsoft's 50th anniversary event, the company announced new features for Microsoft Copilot, including a memory feature that allows Copilot to remember past conversations. With user permission, Copilot will learn about users' preferences, personal details, and work projects to provide more personalized assistance. Microsoft also showcased recent additions to Copilot like vision capabilities and voice mode. Additionally, Microsoft demonstrated an AI-generated version of Quake using their Muse AI model, where every frame of the game is generated by AI in real-time. While some game developers criticized this approach, John Carmack, the original creator of Quake, defended it as a natural progression of technology.
