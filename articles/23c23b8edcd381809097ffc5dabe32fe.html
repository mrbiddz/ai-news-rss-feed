Windows 11 Gets New AI Features

Published: Sun, 27 Jul 2025

Windows 11 has received several new AI features including Recall (search history for your computer), Click to-do (additional AI features on right-click), improved Windows search, and Copilot Vision which provides interactive tutorials by highlighting areas on screen.
