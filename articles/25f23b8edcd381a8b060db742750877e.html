Anthropic Testing Chrome Extension for AI Browser Control

Published: Sun, 31 Aug 2025

Anthropic is testing a Chrome extension that allows AI to directly control a user's browser, including moving the mouse, typing, navigating websites, and making purchases. Unlike OpenAI's Operator which uses a cloud browser, Anthropic's approach works directly in the user's actual browser. The feature is currently limited to 1,000 beta testers with a waitlist available for those interested in trying it.
