Leonardo Releases Lucid Origin Model and Image-to-Video Feature

Published: Sun, 10 Aug 2025

Leonardo has released a new image generation model called Lucid Origin, which offers aesthetics reminiscent of MidJourney V4. The model can create various styles from painterly to comic book-like art and realistic images. Additionally, Leonardo has added an image-to-video feature to its V3 video generation capabilities, allowing users to start with an existing image as the first frame of a generated video, expanding beyond the previous text-to-video-only functionality.
