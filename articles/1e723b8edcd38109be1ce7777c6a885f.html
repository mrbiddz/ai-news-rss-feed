Recraft Introduces New Style Library for AI Image Generation

Published: Sat, 03 May 2025

Recraft, an AI image generation platform, has launched a new style library feature. Users can now browse through an extensive collection of styles in a feed-like interface, apply them to their prompts, and save favorites for future use. The platform also introduced a style blending capability that allows users to combine multiple styles with custom weighting to create unique, personalized styles. These custom styles can be saved and shared with others via links. The new features aim to help users rapidly iterate through different styles and maintain brand consistency across generated images.
