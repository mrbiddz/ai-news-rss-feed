OpenAI Launches GPT Realtime API for Voice Agents

Published: Sun, 31 Aug 2025

OpenAI has introduced GPT Realtime, a new API for developers to create production voice agents. This feature makes the voice conversation capabilities from the ChatGPT app available through the API, allowing developers to integrate similar functionality into their own applications.
