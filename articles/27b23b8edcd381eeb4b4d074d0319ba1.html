Meta Launches Horizon Studio for AI-Generated VR Worlds

Published: Sun, 28 Sep 2025

Meta has introduced Horizon Studio, which allows users to prompt any virtual world into existence within VR. Users can generate environments with text prompts like "ancient ruins" and continue adding elements like "treasure and trinkets" or "mythical guardians" to build interactive AI-generated worlds.
